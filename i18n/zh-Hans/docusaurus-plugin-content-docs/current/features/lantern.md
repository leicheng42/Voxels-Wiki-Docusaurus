# 灯笼

## 灯笼

在您的地块中发出光的功能。光的颜色可以自定义。
此功能是在版本v4.6.0中添加的。
**要使灯笼正常工作，您必须在您的地块上激活[光映射](/docs/Parcels/light-map)**

:::信息
灯笼仍然处于测试阶段。
:::
